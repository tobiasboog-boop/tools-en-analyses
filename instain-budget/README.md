# Instain Budgetanalyse

Interactief dashboard voor budgetbewaking van het Rentree-onderhoudscontract.

## Gebruik

1. Installeer dependencies: `pip install -r requirements.txt`
2. Plaats het Excel-bestand in de `data/` map
3. Start de app: `streamlit run app.py`

## Wachtwoord

Stel een wachtwoord in via Streamlit Cloud Secrets:
```toml
PASSWORD = "je_wachtwoord"
```
